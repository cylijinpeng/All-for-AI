# Roadmap

## Phase 1: Foundation

- Define repository structure
- Publish curation rules and templates
- Seed the first five core collections
- Add basic metadata validation

## Phase 2: Scale

- Add more skills and prompts
- Expand MCP and workflow coverage
- Generate category indexes from metadata
- Add better link checking and linting

## Phase 3: Discoverability

- Add featured collections
- Add scenario-based navigation
- Publish a GitHub Pages site
- Add contributor analytics and monthly updates

## Phase 4: Community

- Accept more external maintainers
- Add discussions for curation decisions
- Publish contribution sprints and themed collections
