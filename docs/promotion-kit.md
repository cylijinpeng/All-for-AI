## Longer launch post

I kept seeing the same problem in AI repos:

- too many generic link dumps
- not enough practical, reusable assets
- no clear answer to “what should I actually use to build X?”

So I built **AI Builder Kit**:

https://github.com/cylijinpeng/ai-builder-kit

It combines:

- curated high-signal tools and frameworks
- prompts in `.prompt.yml` format
- reusable `SKILL.md` assets
- workflow playbooks
- starter packs for coding agents, research assistants, MCP automation, RAG, evals, and local models

If you build with AI, it should save you a lot of time.
