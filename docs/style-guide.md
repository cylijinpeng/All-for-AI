# Style Guide

## Writing

- Use clear, direct language
- Prefer short paragraphs
- Explain why a resource matters
- Avoid hype and vague adjectives

## Formatting

- Use sentence case headings when possible
- Use lowercase kebab-case for folders
- Keep README files readable without special tooling

## Summaries

Good summary:

> Starter MCP setup for browser-driven research tasks with notes on tool boundaries.

Bad summary:

> Amazing next-generation solution for all your browser AI needs.
