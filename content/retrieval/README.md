# Retrieval

Practical retrieval and RAG infrastructure for data-connected AI systems.

## Starter entries

- [`ragflow`](ragflow/README.md): end-to-end RAG engine and app platform
- [`qdrant`](qdrant/README.md): vector database focused on semantic search and filtering
- [`weaviate`](weaviate/README.md): vector database and knowledge retrieval platform
- [`milvus`](milvus/README.md): high-scale vector database for AI search workloads
- [`pgvector`](pgvector/README.md): vector search inside PostgreSQL
