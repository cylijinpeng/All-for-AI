# Inspect AI

## Summary

Open-source framework for evaluating language models and agent systems with strong support for benchmarks and custom tasks.

## Best for

- systematic evaluation workflows
- benchmark-style testing
- agent and LLM task evaluation

## Why it matters

High-signal evaluation framework if you want more structured evaluation than ad hoc prompt testing.
