# Prompt Red Team Skill

## Summary

Skill for stress-testing prompts and agent instructions against failure modes, ambiguity, and abuse.

## Best for

- builders who need a reusable task playbook
- agent workflows that should be more consistent
- turning vague requests into structured outputs

## Suggested behavior

- Identify failure modes
- Create adversarial test cases
- Look for ambiguity and hidden assumptions
- Recommend defensive prompt or eval changes
- Summarize residual risk
