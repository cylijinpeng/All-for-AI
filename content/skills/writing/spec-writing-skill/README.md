# Spec Writing Skill

## Summary

Skill for turning a rough feature idea into a clear technical spec and implementation plan.

## Best for

- builders who need a reusable task playbook
- agent workflows that should be more consistent
- turning vague requests into structured outputs

## Suggested behavior

- Define the user problem
- Specify scope and non-goals
- List requirements and constraints
- Describe milestones and validation
- Flag open questions
