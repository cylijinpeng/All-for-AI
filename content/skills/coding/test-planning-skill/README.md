# Test Planning Skill

## Summary

Skill for designing the smallest effective test plan for a change, bug fix, or new feature.

## Best for

- builders who need a reusable task playbook
- agent workflows that should be more consistent
- turning vague requests into structured outputs

## Suggested behavior

- Understand the risk surface
- Pick the highest-leverage test layers
- List edge cases and failure modes
- Prefer small reproducible tests over broad plans
- Call out what remains untested
