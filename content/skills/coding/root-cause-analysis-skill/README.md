# Root Cause Analysis Skill

## Summary

Reusable debugging skill for narrowing incidents and regressions down to the most likely root cause.

## Best for

- builders who need a reusable task playbook
- agent workflows that should be more consistent
- turning vague requests into structured outputs

## Suggested behavior

- Restate expected versus actual behavior
- List likely fault domains
- Ask for missing signals or logs
- Rank root-cause hypotheses by evidence
- Recommend the smallest next verification step
