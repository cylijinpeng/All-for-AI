Reusable agent behaviors and task playbooks.

## Original starter skills

- [`coding/code-review-skill`](coding/code-review-skill/README.md): review code changes and surface prioritized findings
- [`coding/repo-triage-skill`](coding/repo-triage-skill/README.md): inspect a repository and define the next useful work items

- [`coding/test-planning-skill`](coding/test-planning-skill/README.md): design the smallest effective test plan
- [`research/literature-review-skill`](research/literature-review-skill/README.md): synthesize sources into a structured review
- [`retrieval/rag-audit-skill`](retrieval/rag-audit-skill/README.md): audit retrieval pipelines and prioritize improvements
- [`writing/spec-writing-skill`](writing/spec-writing-skill/README.md): turn ideas into scoped technical specs
- [`evaluation/prompt-red-team-skill`](evaluation/prompt-red-team-skill/README.md): stress-test prompts and instructions
- [`architecture/tool-selection-skill`](architecture/tool-selection-skill/README.md): choose tools and frameworks from concrete constraints
