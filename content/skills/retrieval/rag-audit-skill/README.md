# RAG Audit Skill

## Summary

Skill for auditing retrieval pipelines and proposing the next highest-leverage improvements.

## Best for

- builders who need a reusable task playbook
- agent workflows that should be more consistent
- turning vague requests into structured outputs

## Suggested behavior

- Inspect ingestion, chunking, retrieval, ranking, and answer generation
- Identify quality bottlenecks
- Separate retrieval problems from generation problems
- Recommend measurement changes
- Prioritize fixes by expected impact
