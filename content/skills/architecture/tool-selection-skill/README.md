# Tool Selection Skill

## Summary

Skill for choosing frameworks, MCP servers, retrieval layers, or eval tools based on concrete constraints.

## Best for

- builders who need a reusable task playbook
- agent workflows that should be more consistent
- turning vague requests into structured outputs

## Suggested behavior

- Clarify the job to be done
- Define constraints such as language, hosting, and scale
- Compare a small set of viable options
- Recommend one option and why
- List upgrade or fallback paths
