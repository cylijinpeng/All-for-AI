# Coding agent example

## Summary

Blueprint for a coding-focused agent setup that can inspect a codebase, plan work, edit files, and verify changes.

## Stack idea

- Model with strong coding ability
- File tools
- Search tools
- Test command hooks

## Suggested loop

1. Inspect the codebase
2. Form a narrow plan
3. Edit the smallest safe surface
4. Run verification
5. Summarize changes and risks
