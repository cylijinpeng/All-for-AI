# Research agent example

## Summary

Blueprint for an agent that gathers sources, extracts claims, and writes a concise brief for a human reader.

## Stack idea

- Search or browsing tools
- Structured note extraction
- Citation-aware summarization
- Final brief generation

## Suggested loop

1. Clarify the question
2. Gather candidate sources
3. Rank and filter
4. Extract claims and evidence
5. Write the brief with uncertainties called out
