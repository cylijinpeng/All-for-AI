# MCP Server Design prompt

## Summary

Prompt for scoping an MCP server around a concrete tool or data source.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- Server goals
- Tool surface
- Safety constraints
- First implementation scope
