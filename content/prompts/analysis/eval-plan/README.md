# Eval Plan prompt

## Summary

Prompt for designing an evaluation plan for prompts, agents, or LLM application quality.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- What to measure
- Dataset ideas
- Eval types
- Pass/fail criteria
