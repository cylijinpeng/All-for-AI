# Feature Spec prompt

## Summary

Prompt for turning a rough feature idea into a clear functional and technical spec.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- Problem
- User and use cases
- Scope and non-goals
- Implementation outline
