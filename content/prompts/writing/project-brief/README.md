# Project brief prompt

## Summary

Prompt for shaping a rough AI product or open source idea into a clear project brief.

## Use cases

- Repo planning
- Hackathon scoping
- Agent product definition

## Output expectation

The response should define target user, use case, scope, success metric, and the first build milestone.
