# User Story Pack prompt

## Summary

Prompt for turning a product idea into user stories, acceptance criteria, and release slices.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- Primary stories
- Acceptance criteria
- Edge cases
- MVP slice
