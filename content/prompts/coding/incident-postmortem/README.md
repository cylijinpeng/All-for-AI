# Incident Postmortem prompt

## Summary

Prompt for turning incident notes into a concise postmortem with root cause and prevention actions.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- Impact
- Root cause
- Contributing factors
- Prevention actions
