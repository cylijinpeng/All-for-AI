# Architecture Review prompt

## Summary

Prompt for reviewing a system design or repository architecture and surfacing the most meaningful risks.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- Key strengths
- Highest-risk weaknesses
- Likely scale or reliability issues
- Suggested next design changes
