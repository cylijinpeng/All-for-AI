# Repository audit prompt

## Summary

Prompt for auditing a repository's structure, quality signals, and likely improvement areas.

## Use cases

- Evaluate an open source project quickly
- Produce a technical due diligence note
- Decide whether a repository is ready for contributors

## Output expectation

The response should include repo purpose, strengths, risks, missing pieces, and the next recommended action.
