# Migration Plan prompt

## Summary

Prompt for planning a safe migration across APIs, frameworks, data models, or architecture boundaries.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- Migration phases
- Compatibility risks
- Rollback plan
- Validation checkpoints
