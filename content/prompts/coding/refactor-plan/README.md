# Refactor Plan prompt

## Summary

Prompt for converting messy code or a broad cleanup request into a staged refactor plan.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- Refactor goals
- Constraints
- Step-by-step plan
- Risk controls and test plan
