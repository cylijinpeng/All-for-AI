# Test Strategy prompt

## Summary

Prompt for designing the smallest effective test plan for a change or feature.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- Risk surface
- Recommended test layers
- Key cases
- What can stay manual
