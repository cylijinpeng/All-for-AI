# Deep Research Brief prompt

## Summary

Prompt for turning a broad topic into a focused research brief with questions, sources, and synthesis goals.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- Scope
- Research questions
- Source priorities
- Brief structure
