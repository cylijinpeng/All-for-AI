# Paper summary prompt

## Summary

Prompt for turning a paper or article into a short builder-oriented summary.

## Use cases

- Paper reading notes
- Research brief preparation
- Extracting implementation ideas from publications

## Output expectation

The response should explain problem, method, key findings, limitations, and transferable ideas.
