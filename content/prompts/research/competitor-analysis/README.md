# Competitor Analysis prompt

## Summary

Prompt for comparing products, projects, or approaches and extracting meaningful positioning differences.

## Use cases

- turning vague requests into structured outputs
- giving coding or research agents a clearer task frame
- speeding up early planning work

## Output expectation

The response should include:

- Comparison axes
- Strengths and weaknesses
- Positioning gaps
- Strategic takeaways
