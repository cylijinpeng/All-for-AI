# Prompts

Prompts that can be copied into GitHub Models, IDE agents, or custom apps.

## Starter prompts

- `coding/debug-bug`: turn a bug report into a debugging plan
- `coding/repo-audit`: inspect a repository and produce an audit
- `coding/pr-review`: review a patch with a code-review mindset
- `research/paper-summary`: turn a paper into an implementation-oriented summary
- `writing/project-brief`: shape a rough idea into a project brief
