# Workflows

End-to-end agent flows for common builder tasks.

## Starter workflows

- [`research-to-brief`](research-to-brief/README.md): convert open-ended research into a concise brief
- [`coding/bug-report-to-patch`](coding/bug-report-to-patch/README.md): go from bug report to safe patch
- [`coding/repo-audit-to-issues`](coding/repo-audit-to-issues/README.md): audit a repository and produce a backlog
- [`coding/issue-to-pr`](coding/issue-to-pr/README.md): turn a scoped issue into an implementation and PR summary
- [`coding/spec-to-implementation`](coding/spec-to-implementation/README.md): convert a spec into a validated implementation
- [`evaluation/prompt-to-eval-loop`](evaluation/prompt-to-eval-loop/README.md): improve prompts with explicit evaluation loops
- [`architecture/framework-selection`](architecture/framework-selection/README.md): choose a framework from concrete constraints
- [`retrieval/rag-audit-to-improvements`](retrieval/rag-audit-to-improvements/README.md): convert a RAG audit into prioritized improvements
- [`automation/mcp-server-selection`](automation/mcp-server-selection/README.md): choose or scope an MCP server
