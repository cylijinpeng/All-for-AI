# RAG Audit to Improvements Workflow

## Goal

Workflow for auditing a retrieval system and converting findings into a prioritized improvement plan.

## Steps

1. Inspect ingestion, chunking, retrieval, ranking, and answer generation
2. Separate retrieval errors from generation errors
3. Identify measurement gaps
4. Prioritize the highest-impact fixes
5. Define a re-evaluation plan

## Deliverables

- A concrete output artifact
- A short validation note
- A clear next step if more work is needed
