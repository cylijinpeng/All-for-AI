# MCP Server Selection Workflow

## Goal

Workflow for choosing or scoping an MCP server for a concrete automation or tool-use task.

## Steps

1. Define the task and tool boundary
2. Search for existing MCP servers
3. Evaluate fit, safety, and maintenance
4. Choose an existing server or define a new one
5. Plan validation with MCP Inspector or a client

## Deliverables

- A concrete output artifact
- A short validation note
- A clear next step if more work is needed
