# Bug report to patch workflow

## Goal

Turn a vague bug report into a verified patch with minimal unnecessary change.

## Inputs

- Bug report
- Repository context
- Reproduction environment

## Steps

1. Clarify expected versus actual behavior.
2. Reproduce the bug.
3. Inspect likely files and systems.
4. Make the smallest safe fix.
5. Verify with tests or manual checks.
6. Summarize root cause and residual risk.

## Deliverables

- Reproduction notes
- Patch
- Verification notes
