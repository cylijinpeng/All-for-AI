# Prompt to Eval Loop Workflow

## Goal

Workflow for iterating on prompts with explicit evaluation instead of subjective trial and error.

## Steps

1. Draft or update the prompt
2. Define representative test cases
3. Run the prompt against the cases
4. Review failures and patterns
5. Revise the prompt and repeat

## Deliverables

- A concrete output artifact
- A short validation note
- A clear next step if more work is needed
