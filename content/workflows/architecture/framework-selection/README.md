# Framework Selection Workflow

## Goal

Workflow for choosing an agent or AI framework based on language, constraints, and operational needs.

## Steps

1. Define the job to be done
2. List hard constraints
3. Compare a small set of realistic options
4. Recommend one default choice
5. Define an upgrade or fallback path

## Deliverables

- A concrete output artifact
- A short validation note
- A clear next step if more work is needed
