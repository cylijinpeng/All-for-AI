# mcp-use

## Summary

Open-source library for connecting any LLM to any MCP server and building custom agents with tool access.

## Best for

- custom MCP-powered agents
- connecting multiple MCP servers into one agent stack
- building beyond closed-source host applications

## Why it matters

Useful bridge when you want to work with MCP programmatically instead of only through IDEs or hosted clients.
