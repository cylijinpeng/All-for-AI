# Deep Research Pack

## Summary

Starter pack for source gathering, synthesis, literature review, and research brief generation.

## Best for

- You need a research assistant that can gather sources, compare claims, and write decision briefs.
- You care about citations and synthesis quality.

## How to use this pack

1. Read the linked category pages for the tools in this pack.
2. Start with the smallest viable stack.
3. Add eval and observability before the system gets complicated.
4. Upgrade pieces only when your bottleneck is clear.
