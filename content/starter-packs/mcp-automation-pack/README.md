# MCP Automation Pack

## Summary

Starter pack for tool-using agents built around MCP servers, SDKs, and debugging tools.

## Best for

- You want agents to interact with browsers, files, APIs, or internal systems through a common protocol.
- You need a practical path from zero MCP knowledge to a working tool-using agent.

## How to use this pack

1. Read the linked category pages for the tools in this pack.
2. Start with the smallest viable stack.
3. Add eval and observability before the system gets complicated.
4. Upgrade pieces only when your bottleneck is clear.
