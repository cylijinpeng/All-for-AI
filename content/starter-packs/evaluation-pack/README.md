# Evaluation Pack

## Summary

Starter pack for prompt, agent, and application quality testing.

## Best for

- You want to stop shipping prompt changes blindly.
- You need a minimal but serious quality loop for AI features.

## Recommended stack

- promptfoo for fast comparisons and red teaming
- OpenAI Evals or Inspect AI for more structured evals
- Langfuse or Phoenix for tracing and analysis
- Prompt-red-team and eval-design skills for custom evaluation work

## How to use this pack

1. Read the linked category pages for the tools in this pack.
2. Start with the smallest viable stack.
3. Add eval and observability before the system gets complicated.
4. Upgrade pieces only when your bottleneck is clear.
